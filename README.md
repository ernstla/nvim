# Neovim config

## Requirements

### Tree Sitter

    brew install tree-sitter-cli

### Spectre and Oxi

To use oxi for replacement run

    ~/.local/share/nvim/site/pack/core/opt/nvim-spectre/build.sh

See spectres README fo more information
